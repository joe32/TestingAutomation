1. downlaod the zip file
2. expand it and find the "SETUP" folder inside, open it.
3. double click the "First install-run-ONCE.command" file (only ever run this once, never again). **(it will popup saying it cant be opened) press DONE**
<img width="372" height="378" alt="image" src="https://github.com/user-attachments/assets/e0b0c88c-5eda-42bd-966d-2ce6a61d0115" />
4. open your System Settings app and go to privacy and security, scroll to the bottom and press "open anyway" on the section that says it was blocked.
<img width="835" height="1061" alt="image" src="https://github.com/user-attachments/assets/d621b867-6f32-4ec6-9d30-7701ae1814fe" />
<img width="464" height="101" alt="image" src="https://github.com/user-attachments/assets/e442bab9-21a9-4b38-a39b-366d38be1e42" />
5. repeat these steps for the "Start Testing.command" file (the system settings part only happens on the first time), this will start the tool and open the ui dashboard.
6. in all future instances just double click the "Start Testing.command" whenever you want to run the tool.
7. press "Trigger Run" start the testing and open the bullet suite
<img width="1624" height="1061" alt="image" src="https://github.com/user-attachments/assets/4ac210f8-bb99-4bbf-83af-4a51c857d09c" />
8. login and go to the desired project to test. Then press "start tests" in the bottom right hand corner.
9. check status in the UI dashboard we opened earlier
